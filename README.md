askkk
=====

ask kk = ask 科科
名字好難想喔

Function
--------

### Stage 1

1. 使用者
  1. 透過 invitation 註冊
  2. 登入
  3. 連署問題
  4. 提出問題
  5. 對回應評分, Y/N.
2. 候選人
  1. 登入
  2. 回應問題
  3. 方便看到超過門檻他必須回答的問題
3. 一個回應可以回應多個問題
  * 參考 <https://petitions.whitehouse.gov/response/balanced-approach-reforming-postal-service> 底下列出這個回應所希望回應的問題.

### Stage 2

1. 使用者
  1. 註冊
  2. 在問題底下留言.
  3. 對留言評分, Y/N.
