關於「市長，給問嗎？」

2014 年，公民參與要創造出新的選舉文化。

我們不要傳統的口水戰、文宣戰、旗海戰；
不依賴投票前兩個禮拜才舉辦的政見辯論會；
不單單只是憑藉著黨籍、省籍、性別、或者外表，就決定要投給誰；
更不當只負責投票的「一日公民」。

2014 年，你不必是媒體代表、知名人士，才能對候選人提問。
每一個市井小民，都有一樣的發言權；
當你說出了市民共同的心聲，候選人就必須嚴肅看待、認真回應。

提問和回應都將被公開檢視與討論。
有別於過往的電話或投書反應，
一個公開的平台，
讓市民聲音得以串聯、共鳴，形成更大的力量。

我們相信，你有權利也有義務參與這一次的討論，選出最符合你價值理念的候選人。
因為政治不是「政治人物」或「社運人士」的事，
政治關乎你我的生活，政治是我們的事。

如何使用？——市民指南
1. 提出問題

每位註冊的使用者都可以對候選人提問：你可以選擇詢問一位候選人、或者數位，當然，你也可以詢問所有候選人。

爲維持良好的討論品質，請務必閱讀並記得遵守版規。

2. 開始集氣

提問送出後，每則提問都有一個月的時間可以被連署（被推、尋找共同發問者）。

連署一個問題，代表你也關注這個議題，並認為候選人對此提出看法及回答是重要的。

3. 達成門檻

當一個提問在時間內達到1,000連署數（被推爆了），候選人就必須回答此問題。

4. 候選人回答

針對候選人的回答，民眾可以投票支持（推）或反對（噓）。

回覆後，便無法修改回覆內容；若有不足或想補充說明之處，可針對同一問題再次回答。



如何使用？——候選人指南
1. Inbox / 市民心聲

候選人登入後，可以看到專屬於自己的「市民心聲」信箱，所有向此候選人提問且已達門檻的問題會出現在這裡。

你可以想像成這是你的 gmail 信箱，當連數人數達到門檻後，系統就會自動把這個問題寄到你的信箱。

2. 已回覆 / 未回覆

在「市民心聲」信箱中，可以分別看到「已回覆」和「未回覆」的列表，方便候選人檢視還有哪些問題尚未回覆。

3. 回覆問題

回覆問題很簡單：在每個提問下方都有一欄輸入文字的欄位，候選人只需在此輸入回應後點選「回覆」，就可以完成回覆了。

4. 多次回覆

候選人回覆後，便無法修改回覆內容；若有不足或想補充說明之處，可針對同一問題再次回答。

和其他平台有什麼不同？

不管是選舉或是和政治人物溝通的方式，我們都不是唯一、也不是第一。但我們和其他的管道或平台有什麼不同呢？

旗海宣傳

候選人對選民的單向宣傳，無法雙向溝通。

市長，給問嗎？

候選人可以從平台上獲得選民意見，選民也可得知候選人回答，是候選人和選民的雙向溝通平台。

媒體民調

只有媒體才能決定民調的時間點和調查項目。

市長，給問嗎？

平台上的「連署」和「針對候選人回應投票」的機制，就像是一場即時民調；任何人都可以丟出自己關注的議題。

政見辯論

由各領域代表負責提問，沒有機制讓一般民眾參與。

市長，給問嗎？

所有人都可以提問，代表是動態、即時被票選出的，真實說出市民心聲的人將成為市民代表。

選民陳情

目前民眾可以透過電話、網路等方式陳情，也會獲得回應，但是是私下的聯絡，其他人無法得知「誰」和「誰」反應了什麼問題，陳情的人也不會知道大概有多少人和自己會有一樣的問題。

市長，給問嗎？

在公開的平台上連署，達到陳情的效果，同時凝聚共識，讓關注同樣議題的人可以在平台上找到彼此，共同發聲。
